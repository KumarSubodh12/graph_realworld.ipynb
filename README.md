# ADA Lab Experiment 3

## Graph Algorithms Mini Project

---

### 📌 Description

This project focuses on the implementation of fundamental graph algorithms used in real-world applications such as network routing, task scheduling, and infrastructure design. It covers graph traversal, shortest path computation, topological sorting, and minimum spanning tree algorithms.

---

### ⚙️ Algorithms Implemented

#### 1. Graph Representation

* Adjacency Matrix
* Adjacency List
* Used to efficiently represent graph structures

---

#### 2. Graph Traversal

* Breadth First Search (BFS)
* Depth First Search (DFS)
* Used for exploring connectivity and searching nodes

---

#### 3. Topological Sorting

* Applied on Directed Acyclic Graph (DAG)
* Used in task scheduling and dependency resolution

---

#### 4. Shortest Path Algorithms

* Dijkstra’s Algorithm
* Computes shortest distance from source node
* Used in navigation and routing systems

---

#### 5. Minimum Spanning Tree (MST)

* Prim’s Algorithm
* Finds minimum cost to connect all nodes
* Used in network design and cabling

---

### 📊 Performance & Analysis

* Execution time measured for algorithms
* Observations added in notebook
* Optional visualization using matplotlib
